# Wealth Matrix AI — Shorts Production Standard (v1)

A reusable checklist and master reference document for every future video generated by the factory. All scripts, Prompts, and React code must adhere strictly to these rules.

## 1. Master Timing Blueprint (~42–45s total)

| TimeBlock     | Purpose                 | Details                                                                                  |
|---------------|-------------------------|------------------------------------------------------------------------------------------|
| 0:00–0:05     | Hook                    | Bold claim or question. No unsourced institution name-drops.                             |
| 0:05–0:09     | Suspense banner         | e.g. "Wait for #3" — label must logically match the payoff card later.                   |
| 0:09–0:14     | Authority / Quote       | Presenter avatar delivers one strong opinion line.                                       |
| 0:14–0:19     | Point 1                 | Insight card.                                                                            |
| 0:19–0:24     | Point 2                 | Insight card.                                                                            |
| 0:24–0:29     | Point 3                 | Data/stat card — only if you have a real, checkable source.                              |
| 0:29–0:32     | Tagline                 | One-line emotional pivot.                                                                |
| 0:32–0:36     | Case Study              | Real-world example card.                                                                 |
| 0:36–0:41     | Recap                   | Must include every point shown above — no silent omissions.                              |
| 0:41–0:45     | Outro                   | Logo + CTA.                                                                              |

## 2. Hook & Banner Rules
- **No Fake Data**: Never attribute a number to a real institution ("Harvard," "RBI," "Forbes," etc.) unless naming and linking the actual source. Reframe as a belief: "Many believe..." / "It's often said that...".
- **Curiosity Payoff Consistency**: Whatever the suspense banner promises ("Wait for #3"), the later payoff card must match.

## 3. Credibility / Sourcing Rules
Use this badge hierarchy consistently:
- **💡 INSIGHT** — opinion or framework. No source needed.
- **📚 CASE STUDY** — a real example, paraphrased. A loose source mention is fine.
- **📊 DATA** — an actual number. Only use this badge if there's a real, linkable source behind it.
- **✅ VERIFIED** — reserve this only for claims checked against a primary source. Never put it on a paraphrase or a vague claim about a real person.
- **Naming Protocol**: When naming a real public figure, use a full, correct name once (e.g., "Dhirubhai Ambani") and stick to well-documented public facts.

## 4. Typography Standards
- **Spacing**: No space before punctuation. 
  - Correct: Rule 1: Never... / safety; optimize... / ownership.
  - Incorrect: Rule 1 : Never... / safety ; / ownership .

## 5. Recap Rules
- The recap card must mirror 100% of the points shown earlier — same count, same order. If a point gets cut from the recap, cut it from the main video too.
- Don't restate the label inside the line itself (e.g. "1. Rule 1: Never..."). The number badge already does that job — just state the point once.

## 6. Branding Rules
- Keep the small round logo watermark visible in a corner for the ENTIRE video, not just the hook.
- Keep the "Wealth Matrix AI" bottom-right wordmark visible throughout.

## 7. Language Policy
- Pure Hindi / Hinglish / English on-screen text MUST match the voiceover language. 
- For a Hindi-first audience, **Hinglish** captions (Hindi sentence rhythm + English finance terms, like "SAVE KARO") is the default register.

## 8. Pre-Publish QA Checklist (For Autonomous Agent)
- [ ] Every named source is real and verifiable, or rewritten as a belief/opinion
- [ ] Banner promise label = payoff card label, word for word
- [ ] No space before any colon, semicolon, or period anywhere on screen
- [ ] Recap includes every point from the video, in the same order, nothing dropped
- [ ] No repeated label-in-line phrasing in the recap
- [ ] Logo watermark visible from start to finish
- [ ] On-screen text language matches the intended audience and voiceover
- [ ] Voiceover and captions are roughly in sync, beat for beat
